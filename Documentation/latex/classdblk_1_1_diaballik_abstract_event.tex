\doxysubsection{Detailed Description}
The Diaballik\+Abastract\+Event class. 

This class represents a main action launched by the user. It can be a passive action that dont modifies the game data (like ask for help, refresh the board...) or an active action who modifies the game data (move a piece, pass turn...).
